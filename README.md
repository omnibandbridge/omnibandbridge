# OmniBandBridge (OBB)
Free your Mi Band. OmniBandBridge (OBB) streams real-time footpod data (cadence, pace, running power) to Zwift and exports your workouts. It also handles treadmill speed &amp; incline (FTMS sensor or manual) — usable even purely as an FTMS/treadmill bridge. Public community &amp; issue tracker: bugs, feedback and feature requests.
